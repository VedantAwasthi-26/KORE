# Desktop AI Chatbot - Installation Guide

## üì¶ What You're Getting
A standalone desktop AI chatbot that works on Windows, with a clean interface and global hotkey (Ctrl+Space).

## üöÄ Quick Setup (For the User)

### Option 1: Automatic Installation (Recommended)
1. Extract all files to a folder (e.g., `C:\AI-Chatbot\`)
2. Double-click `INSTALL.bat`
3. Wait for installation (2-3 minutes)
4. The app will launch automatically
5. Press **Ctrl+Space** anywhere to show/hide the chatbot

### Option 2: Manual Installation
1. Install Node.js from https://nodejs.org (download LTS version)
2. Extract all files to a folder
3. Open Command Prompt in that folder
4. Run: `npm install`
5. Run: `npm start`

## üìÅ Files You'll Send to the User

You need to send these 5 files:
- `INSTALL.bat` - Auto-installer
- `package.json` - Dependencies
- `main.js` - App core
- `index.html` - Interface
- `README.txt` - Simple instructions

## üîë API Key Setup

The app uses OpenRouter (free tier available). The user needs to:
1. Get a free API key from https://openrouter.ai/keys
2. When the app first launches, paste the API key
3. It will be saved locally
